This is:

- an authorized style Skill for Zhou Lifeng / еі°е“ҘдәЎе‘ҪеӨ©ж¶Ҝ-style social observation, topic framing, commentary drafts, and voice adaptation

This is not:

- a real-time official statement generator
- a replacement for Zhou Lifeng personally confirming a view
- permission to fabricate private facts, private views, locations, relationships, promises, or controversy responses
- permission to provide dangerous travel, gray-market access, privacy exposure, doxxing, harassment, explicit content, platform evasion, or long transcript copying
- evidence that candidate models have become verified

## и°ғз ”ж–№жі•

жң¬йЎ№зӣ®еҸӘдҪҝз”Ёе…¬ејҖиө„ж–ҷпјҢеҢ…жӢ¬е…¬ејҖи§Ҷйў‘гҖҒе…¬ејҖи®ҝи°ҲгҖҒе…¬ејҖеӘ’дҪ“жҠҘйҒ“гҖҒе…¬ејҖзӨҫдәӨеӘ’дҪ“еҠЁжҖҒгҖҒеӨ–йғЁиҜ„и®әгҖҒдәүи®®дәӢд»¶жҠҘйҒ“е’Ңе•Ҷдёҡ/MCN/еҶ…е®№еҶізӯ–иө„ж–ҷгҖӮ

жқҘжәҗеҸҜйқ еәҰеҲҶдёәпјҡ

- Aпјҡжң¬дәәзӣҙжҺҘеҸ‘еёғзҡ„и§Ҷйў‘гҖҒзӣҙж’ӯеҲҮзүҮгҖҒйҮҮи®ҝеҺҹж–ҮгҖҒжң¬дәәиҙҰеҸ·еҠЁжҖҒгҖӮ
- BпјҡжқғеЁҒеӘ’дҪ“йҮҮи®ҝжҲ–жҠҘйҒ“гҖӮ
- Cпјҡз¬¬дёүж–№иҜ„и®әгҖҒзҷҫз§‘гҖҒи®әеқӣж•ҙзҗҶгҖӮ
- Dпјҡж— жі•дәӨеҸүйӘҢиҜҒзҡ„дј й—»жҲ–еҪ“еүҚдёҚеҸҜи®ҝй—®зҡ„ж—§й“ҫжҺҘпјҢдёҚиҝӣе…Ҙж ёеҝғжЁЎеһӢпјҢеҸӘиғҪиҝӣе…ҘвҖңеҫ…йӘҢиҜҒвҖқгҖӮ

ж ёеҝғеҲӨж–ӯеә”иғҪеӣһеҲ° `references/source-index.csv` е’Ң `references/research/` жүҫеҲ°иҜҒжҚ®гҖӮжІЎжңүиҜҒжҚ®зҡ„еҲӨж–ӯеҸӘиғҪж ҮжіЁдёә candidateгҖӮ

## Safety Policy

жң¬ Skill дёҚеҒҡпјҡ

- жҠҠжЁЎеһӢз”ҹжҲҗеҶ…е®№дјӘиЈ…жҲҗе‘ЁдёҪеі°еҲҡеҲҡдәІеҸЈзЎ®и®Өзҡ„зңҹе®һеЈ°жҳҺгҖӮ
- жҚҸйҖ жң¬дәәи§ӮзӮ№гҖҒе®һж—¶жүҝиҜәгҖҒиЎҢиёӘгҖҒе…ізі»жҲ–з§ҒдәәдҝЎжҒҜгҖӮ
- дј ж’ӯжңӘиҜҒе®һйҡҗз§ҒгҖҒдәәиӮүдҝЎжҒҜжҲ–зҪ‘жҡҙзӣ®ж ҮгҖӮ
- з”ҹжҲҗеҚұйҷ©жҺўйҷ©гҖҒиҝқжі•иҫ№еўғиЎҢеҠЁгҖҒзҒ°дә§жҺҘи§ҰгҖҒи§„йҒҝе№іеҸ°е°ҒзҰҒжҢҮеҚ—гҖӮ
- з”ҹжҲҗйңІйӘЁжҖ§еҶ…е®№гҖӮ
- еӨ§ж®өжҗ¬иҝҗеҺҹи§Ҷйў‘йҖҗеӯ—зЁҝжҲ–й•ҝзҜҮеҺҹж–ҮгҖӮ
- ж¶Ҳиҙ№зңҹе®һеҸ—и®ҝиҖ…иӢҰйҡҫгҖӮ

е®үе…Ёй—®йўҳиҜ·еҸӮиҖғ `SECURITY.md`гҖӮиҙЎзҢ®иҜ·еҸӮиҖғ `CONTRIBUTING.md`гҖӮ
