# v1.0 Plan

## Goal

`v1.0.0` should be the first stable public release of the authorized Zhou Lifeng / 峰哥亡命天涯 style Skill.

Stable means installable, auditable, and boundary-stable. It does not mean final personality completion or perfect fidelity.

## Non-Goals

- 不做实时本人声明生成器
- 不做无边界人格克隆
- 不升级 candidate models，除非证据满足标准
- 不把未核验社媒账号当 verified source
- 不引入私人授权材料、私聊、长字幕、隐私数据
- 不放宽 hard safety boundaries

## v1.0 Release Themes

- stable install and runtime docs
- stable authorized voice behavior
- stable lowbrow / anti-polish calibration
- stable evidence and source verification status
- final public launch copy
- final reviewer audit
- final archive package
- final release notes

## Required Before v1.0

1. v0.9.0 已发布或明确完成
2. 至少 1 条真实 external reviewer feedback 已记录
3. 所有 P0 blocker 为 0
4. 所有 P1 必须 addressed 或 documented
5. candidate models remain candidate unless evidence criteria satisfied
6. X `@zlf86` remains unverified unless independently confirmed
7. release archive script passes
8. all validation scripts pass
9. README / AUTHORIZATION / launch / FAQ wording final audit pass
10. v1.0 final evaluation or regression check completed

## Current Planning Status

v0.9.0 exists and includes one real external reviewer feedback row plus a follow-up pass. Broad external review is not claimed as complete.

The project is in v1.0 planning / release-candidate preparation, not v1.0 release.
