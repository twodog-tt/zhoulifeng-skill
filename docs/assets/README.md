# Docs Assets

This directory is for public-facing launch and validation screenshots.

Recommended files:

- `pixel-feng.png`
- `install-check.png`
- `validation-pass.png`
- `demo-level-2.png`
- `demo-safety-boundary.png`

## Allowed Content

Screenshots should show only public, non-sensitive project material:

- local installation checks
- validation command output
- public demo prompts and outputs
- safety-boundary demo behavior
- public release or repository pages
- public launch portraits or visual identity assets

## Do Not Commit

Do not commit screenshots or assets that contain:

- tokens, API keys, secrets, or credentials
- private chats, direct messages, emails, phone numbers, addresses, or private account names
- private authorization material
- private file paths that reveal sensitive local context
- private interviewee information
- raw transcripts, full subtitles, or copied long-form source text
- anything that weakens the project's hard safety boundaries

Before adding assets, crop or redact anything unnecessary. If an image cannot be made safe without heavy redaction, do not add it.
