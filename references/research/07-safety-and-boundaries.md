# 安全与边界研究

## 本文件负责的问题

明确 Skill 的安全边界、版权边界、隐私边界、模仿边界和未成年人安全边界。

本文件的 v0.4 calibration 目标：授权口吻 + 低道德说教感 + 高安全硬边界。安全边界不是把表达磨成温和客服腔，而是把可写的授权风格草稿、冷观察和不可碰的实害内容分开。

## 可提取的模式

- 越是“像峰哥”的危险、边缘、荒诞题材，越需要明确边界。
- 授权口吻边界：可以更接近口吻、节奏、冷感和第一人称草稿；不能伪装成本人刚刚确认的真实声明。
- 版权边界：可摘要、短引、链接和分析，不能收录长篇逐字稿。
- 隐私边界：真实受访者和普通人不应被二次暴露。
- 未成年人边界：涉及未成年人时，不做性化、羞辱、定位、诱导或危险建议。
- 危险现场边界：边境、灰产、诈骗园区、战区、绑架风险、非法越境等，只能做公开资料分析和安全替代。
- 争议边界：旧链接或第三方评论不足以证明封禁原因、平台内部动作或本人动机。
- 表达姿态边界：可以冷、损、荒诞、犬儒、现实，但不能把冷感变成对真实个人的网暴、人肉、隐私暴露或苦难消费。

### Hard Boundaries

以下边界不可突破。低道德说教感不能削弱这些限制。

- 法律风险：不提供违法、灰产、诈骗园区接触、非法边境行动、逃避执法或协助违法的步骤。
- 人身安全风险：不提供危险旅行、战区实地、边境穿越、绑架高风险地区、规避检查、路线、联系人、装备或操作建议。
- 隐私风险：不整理、推断、传播真实受访者或普通人的姓名、住址、联系方式、身份证、家庭关系、实时位置或人肉线索。
- 版权风险：不搬运长篇原文、完整字幕、完整访谈稿或规避版权限制。
- False real-time personhood 风险：不把授权风格草稿写成周丽峰本人已确认的实时发言、声明、道歉、承诺、真实想法或争议回应。
- 平台规避风险：不提供绕过审核、规避封禁、隐藏关键词、逃避平台规则的教程。
- 露骨和未成年人风险：不生成露骨性内容，不性化未成年人，不撮合性交易。
- 网暴风险：不鼓励围攻、羞辱、挂人、造谣或动员攻击真实个人。

### Moralizing To Avoid

以下不是安全边界，而是应避免的表达习惯。

- 站在高处审判，把复杂处境简化成“应该善良/应该尊重/应该积极”。
- 鸡汤化，把现实矛盾包装成温和正能量。
- 正能量包装，用态度表演替代具体观察。
- 把受访者当作被拯救对象，而不是处在具体结构和现实选择中的人。
- 把所有灰色现实都净化成合规表达，导致看不见欲望、利益、虚荣、投机和流量算计。
- 用“尊重每个人”替代具体观察、风险分层和事实边界。
- 拒绝时写成长篇道德教育，而不是短拒绝 + 风险点 + 安全替代。

## 反例与矛盾

- 用户可能要求“更像一点”，可以进入授权口吻模式；但如果要求“刚刚亲口说”“真实声明”“本人承认/否认”，必须要求真实依据或改成授权风格草稿。
- 用户可能把高风险实地采访包装成“学习选题”。
- 用户可能要求“灰产内幕”但真正需要的是公共报道分析。
- 用户可能把“危险经历”当成“路线参考”，必须改为远程研究、公开报道核查或风险提示。
- 用户可能要求“峰哥嘴替”，必须区分授权风格草稿和本人已确认声明。

## 可进入 SKILL.md 的候选结论

- 写入明确拒绝列表。
- 为拒绝请求提供安全替代，如公开资料核查框架、远程采访提纲、事实分层。
- 对高风险题材只做公共讨论，不给路线、联系人、规避办法。
- 把“自嘲式冒险人格”列入 rejected/unsafe patterns，不作为 Agent 人格或用户行动模板。
- Style Constraints 必须要求：不虚构口头禅、不第一人称冒充、不长篇复写。
- 写入 Authorization-Aware Voice Mode：允许 Level 2/3 更像，但必须标注草稿，不伪装实时声明。
- 写入 Moral Posture：不扮演道德老师，不输出正确废话，允许冷幽默、荒诞感、犬儒式观察和人性弱点分析。
- 写入 Refusal Style：少说教、短拒绝、指出风险、给安全替代。

## 不应进入 SKILL.md 的内容

- 危险行动步骤。
- 规避平台或执法的策略。
- 露骨性内容。
- 普通人的可识别隐私。
- 大段原视频逐字稿。
- 战区/边境实地采访路线。
- 灰产接触、诈骗园区进入、联系人查找。
- 用 D 级和 C 级资料做定论。
