# Source Verification

This directory tracks social-account verification status for public source use.

The workflow is especially important for accounts such as X / Twitter `@zlf86`, where public posts may be useful for candidate voice calibration but should not enter verified models unless account ownership and source reliability are independently confirmed.

## Files

- `account-verification-checklist.md`: verification criteria.
- `social-account-status.csv`: current account status table.
- `reports/v0.8-source-verification-report.md`: current v0.8 verification status.

## Rule

Unverified social accounts may be used as search leads or candidate voice calibration material only. They must not be used as verified model evidence.

Do not add private chats, private screenshots, private authorization material, or non-public data here.
