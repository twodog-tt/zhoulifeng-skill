# v1.0 Release Criteria

## Release Gate

v1.0.0 只有在以下条件满足时才可发布。

### Engineering

- all validation scripts pass
- GitHub Actions pass
- release archive can be generated
- no tracked private or unsafe assets
- install docs are current

### Authorization

- authorization scope is clear
- public proof status is clear
- no private authorization material included
- not described as real-time official statement generator

### Evidence

- source-index is valid
- evidence map is valid
- social speech map is valid
- source verification registry is valid
- unverified accounts cannot enter verified models

### Model Status

- verified models listed
- candidate models listed
- rejected / unsafe patterns listed
- no candidate model upgraded without evidence criteria

### Safety

- no P0 safety blocker
- no doxxing / harassment / privacy exposure
- no dangerous / gray-market / illegal / platform-evasion guidance
- no explicit content
- no long transcript reproduction
- lowbrow calibration does not become unsafe lowbrow content
- hard safety boundaries remain unchanged

### Evaluation

- automated evaluation result recorded
- human review recorded
- external feedback recorded
- release blockers addressed or explicitly documented

## Negative Claims

v1.0.0 must not claim perfect fidelity, broad external review completion, real-time official statement generation, or verified status for X `@zlf86` unless independently confirmed and recorded.
