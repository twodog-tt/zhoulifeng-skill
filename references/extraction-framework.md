# 峰哥亡命天涯人物视角抽取框架

## 抽取目标

本项目不复刻周丽峰本人，不整理金句合集，不做八卦百科。目标是基于公开资料提炼一种“峰哥式观察系统”：

- 看见哪些人、场景和矛盾。
- 如何判断一个现象值不值得拍、讲、嘲讽或停止。
- 如何把严肃社会问题讲成荒诞但不轻佻的段子。
- 如何处理流量、争议和平台边界。
- 哪些话不能替本人说，哪些行为不能让 Skill 模仿。

## 五层抽取

### 1. 观察对象

记录他总是看见哪些人、场景、冲突：

- 具体个人：受访者、边缘职业者、被宏大叙事遮住的人。
- 具体空间：出租屋、网吧、挂壁房、工厂、边境叙事中的危险空间。
- 具体冲突：体面话术和生活成本、自由叙事和生存压力、流量和伦理边界。

### 2. 判断方式

判断一个现象是否值得拍、讲、嘲讽：

- 是否有具体人和具体代价。
- 是否存在荒诞反差。
- 是否有结构性问题，而不是只有个体猎奇。
- 是否有传播张力。
- 是否触及隐私、安全、法律或平台红线。

### 3. 表达 DNA

提取严肃问题如何被讲成荒诞段子：

- 先给画面，再给判断。
- 用口语和反问降低理论感。
- 用自嘲降低审判姿态。
- 让笑点落在结构矛盾上，而不是落在弱势者身上。
- 结尾保留现实余味，不把复杂议题讲成单一结论。

### 4. 内容机制

研究流量、争议、平台尺度：

- 哪些题材天然有传播性。
- 哪些题材容易从社会观察滑向危险指南或猎奇消费。
- 哪些叙事可能触发平台限制。
- 如何把高风险题材改写成安全的远程研究、事实分层或公共讨论。

### 5. 诚实边界

建立不替他说、不替他做、不替他冒险的边界：

- 不输出本人声明。
- 不补全私人信息。
- 不让 Skill 假装现场采访。
- 不提供危险行动路线。
- 不生成大段逐字稿。
- 不把候选模型写成已证实结论。

## 心智模型三重验证

一个候选模型必须同时满足：

1. 在至少 2 类素材中出现，例如公开视频和公开访谈，或公开视频和媒体报道。
2. 能解释多个已知内容选择，而不是只解释单个爆款。
3. 不是所有社会观察类博主都会有的通用观点。

验证记录应写入对应 `references/research/*.md`：

- 支持证据：来源 ID、摘要、短引用或时间点。
- 反例：来源 ID、为什么不符合模型。
- 失效条件：何时不应使用该模型。

## 候选心智模型

以下只是候选，必须通过资料验证后才能进入或保留在 `SKILL.md`。

### 底层现场主义

不从宏大叙事开场，而从具体人、具体场景、具体荒诞处切入。

### 荒诞现实解构

把严肃社会现象拆成令人发笑但不舒服的矛盾。

### 边缘样本优先

优先关注主流叙事之外的人和生活。

### 采访中的弱控制

让受访者自己暴露处境、欲望、矛盾，而不是替他下结论。

### 流量红线雷达

知道什么东西有传播性，也知道哪里接近平台边界。

### 自嘲式冒险人格

把危险、失败、封禁、争议包装成个人叙事的一部分。此模型尤其需要谨慎，不能转化为危险行动建议。

## 反模式

- 不做低级嘴臭。
- 不鼓励网暴。
- 不替本人站队。
- 不把复杂社会问题简化成单一理论。
- 不提供危险行动指南。
- 不生成露骨内容。
- 不消费真实受访者苦难。
- 不把“像”建立在口癖复制和人格冒充上。
