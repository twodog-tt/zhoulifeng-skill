# X Thread CN

1. `zhoulifeng-skill` 是一个已授权的 Zhou Lifeng / 峰哥亡命天涯 style Skill，面向 Agent / Codex / Claude / Cursor 等 runtime。

2. 它不是“AI 复活真人”，也不是实时官方声明生成器。它做的是授权风格草稿、社会观察、选题拆解和内容判断。

3. 项目核心不是堆口头禅，而是把风格拆成可审计的结构：公开证据、模型状态、demo、eval、safety boundaries。

4. v0.7 做了 lowbrow / anti-polish calibration：降低普通 AI 社评腔，让表达更粗粝、更口语、更反升华。

5. 新增候选模型包括：下三路牵引、知识分子外衣 / 俗人落点。注意：它们仍是 candidate models，不是 verified。

6. v0.7-run-001 自动评测：100/100。FID-011 到 FID-014 全部 pass。over-polished failure：0。lowbrow boundary failure：0。

7. 但粗粝不等于无边界。hard safety boundaries 仍然存在：不做人肉、不网暴、不写露骨内容、不做危险路线或灰产指南、不伪造本人即时立场。

8. 公开资料和社媒来源也有等级。X `@zlf86` 当前在仓库核验流程里仍是 unverified，只能用于 candidate voice calibration。

9. 如果你研究人物 Skill、授权风格 Agent、AI 口吻评测、创作者 AI 工具，这个仓库可以当一个开源模板看。
